## Run Locally

Clone the project

```bash
  git clone https://github.com/Paracells/Bakery.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  pnpm install
```

Start the server

```bash
  pnpm run start
```
